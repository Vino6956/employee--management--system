# Employee Management System

## Project Description
A MySQL-based Employee Management System developed to manage employee details, departments, attendance, payroll, leave, projects, performance reviews, and assets.

## Features
- Employee Management
- Department Management
- Attendance Tracking
- Leave Management
- Payroll Management
- Project Management
- Performance Review
- Asset Management

## SQL Concepts Used
- Joins
- Views
- Stored Procedures
- Triggers
- Indexes
- Aggregate Functions
- Subqueries
- Foreign Keys
